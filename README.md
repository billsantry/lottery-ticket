# NOT A LOTTERY TICKET: 8-BIT FUTURES

A retro arcade strategy game inspired by Peter Thiel's talk *"You Are Not a Lottery Ticket."*

Choose a worldview. Allocate resources. Shape your future across 12 years. Feel why each philosophy leads to its outcomes.

## Worldviews

- **Determinate Optimism (DO):** Specific plans, conviction, concentrated bets, compounding
- **Indeterminate Optimism (IO):** Diversification, A/B testing, process optimization, optionality
- **Determinate Pessimism (DP):** Rationing, fortress strategy, reserves, survival-first
- **Indeterminate Pessimism (IP):** Insurance, hedging, compliance, risk minimization

## Play

Open `index.html` in any browser. No dependencies, no build step, no server required.

- **1-Player:** Solo 12-year strategic run
- **2-Player:** Local hot-seat, shared world events

## Deploy

This is a single static HTML file. Deploy anywhere: GitHub Pages, Azure Static Web Apps, Netlify, S3, or just double-click the file.

## Controls

Mouse clicks or keyboard. Pick actions, end your year, react to events.
